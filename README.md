# Project Description
GameX is a telegram bot that allows you to find discounted games, information about a particular game, upcoming releases, and games that have become completely free for a while on the steam online service

# Features

 🔥 Top Discounts
Shows the best Steam discounts sorted by deal rating.

 🆓 Free Games
Displays games that became temporarily free.

 🚀 Upcoming Games
Shows the most anticipated upcoming releases.

 🔍 Game Search
Users can search any game and get:
- poster/image
- Metacritic score
- genres
- release date
- short description
- Steam page link

 📄 Help 
Built-in help system with feature explanations.

# Technologies used
- Python
- Aiogram 3
- aiohttp
- RAWG API
- CheapShark API
- Telegram Bot API
